# openLog 项目记忆索引

本目录存储 openLog 项目的持久化上下文信息，用于在不同会话和不同电脑之间无缝衔接。

## 记忆文件

- **project_progress.md** — 项目整体进度和架构概览
- **TASK_STATUS.md** — 当前任务状态和待办事项

## 如何使用

新会话开始时：

1. 阅读 `memory/project_progress.md` 了解项目架构和现状
2. 阅读 `memory/TASK_STATUS.md` 了解当前工作进度
3. 阅读 `docs/debug-flow-guide.md` 了解三种使用模式
4. 查看 `docs/superpowers/specs/` 中的设计文档
