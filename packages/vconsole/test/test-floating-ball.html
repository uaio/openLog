测试用例

请按照以下测试用例逐一验证功能，每个测试用例完成后点击"通过"或"失败"按钮。
1. 基础点击激活 待测试
点击屏幕右下角的悬浮球，验证面板是否正常显示；再次点击验证面板是否隐藏。
2. 自由拖拽移动 待测试
按住悬浮球拖拽到屏幕中央位置，松手后验证悬浮球是否停留在该位置（不吸附）。
3. 左侧边缘吸附 待测试
拖拽悬浮球到屏幕左边缘（距离 < 20px），松手后验证：
• 悬浮球吸附到左边缘
• 只显示右半球（clip-path 裁剪）
• 透明度降低至 0.3
4. 右侧边缘吸附 待测试
拖拽悬浮球到屏幕右边缘（距离 < 20px），松手后验证：
• 悬浮球吸附到右边缘
• 只显示左半球（clip-path 裁剪）
• 透明度降低至 0.3
5. 吸附状态单击无效 待测试
在吸附状态下，单击悬浮球，验证面板不会显示（需要三连击才能激活）。
6. 吸附状态三连击激活 待测试
在吸附状态下，500ms 内快速连续点击 3 次，验证：
• 悬浮球从吸附状态恢复
• 恢复完整显示（不再裁剪）
• 面板正常打开
注意：三连击需要在 500ms 内完成，请快速点击。

技术实现

核心组件

  • FloatingBall - 悬浮球主组件，管理状态和生命周期
  • DragHandler - 拖拽处理器，处理鼠标和触摸事件
  • ClickDetector - 连击检测器，检测三连击事件

状态机

  • normal - 正常状态，点击即可激活
  • snapped - 吸附状态，需要三连击激活
  • dragging - 拖拽中，禁用点击事件
